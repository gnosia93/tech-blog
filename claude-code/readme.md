### CLAUD.md ###
CLAUDE.md는 Claude Code가 세션을 시작할 때 자동으로 읽어들이는 프로젝트 메모리 파일이다. 프로젝트에 대해 매번 반복 설명하지 않아도
되도록, Claude가 항상 알고 있어야 할 내용을 적어둔다.

#### 주로 담는 내용 ####

- 프로젝트 개요: 이 코드베이스가 무엇을 하는지, 아키텍처 구조
- 자주 쓰는 명령어: 빌드, 테스트, 린트, 실행 명령 (예: npm run build)
- 코드 컨벤션: 네이밍 규칙, 포맷팅, 선호하는 패턴/라이브러리
- 주의사항: "이 파일은 건드리지 마라", "이 API는 이렇게 써라" 같은 규칙

#### 위치와 우선순위 ####
여러 곳에 둘 수 있고, 아래로 갈수록 더 좁은 범위에 적용된다.
- ~/.claude/CLAUDE.md — 모든 프로젝트에 공통 적용 (개인 전역 설정)
- <프로젝트 루트>/CLAUDE.md — 해당 프로젝트에만 적용 (보통 git에 커밋해서 팀과 공유)
- <하위 디렉토리>/CLAUDE.md — 그 디렉토리 작업 시 추가로 적용
- CLAUDE.local.md — 개인용, 보통 git에 안 올림

#### 만드는 법 ####

/init 명령을 실행하면 Claude가 현재 코드베이스를 분석해서 CLAUDE.md 초안을 자동으로 만들어 준다..
